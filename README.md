# EV Incentive App

A web application designed to help users understand and calculate electric vehicle (EV) incentives, charging costs, and related information.

## Description

This application provides tools for EV owners and potential buyers to:
- Calculate charging costs
- Compare different EV options
- Find nearby dealers
- Explore available schemes and savings
- Access contact information

## Features

- **Charging Cost Calculator**: Estimate the cost of charging your EV
- **Comparison Tool**: Compare different EV models and options
- **Dealer Locator**: Find authorized EV dealers in your area
- **Savings Calculator**: Calculate potential savings with EVs
- **Schemes Information**: Learn about available government schemes and incentives

## Installation

1. Clone this repository to your local machine
2. Navigate to the project directory
3. Open `index.html` in your web browser

## Usage

- Start by opening `index.html` in your browser
- Use the navigation menu to access different sections
- Fill in the forms and calculators as needed
- All functionality is client-side and requires no server setup

## Technologies Used

- HTML5

## Challenges Faced

During the development of this project, we encountered several challenges:

- **Data Accuracy**: Ensuring that all EV-related data, pricing, and incentive information remains current and accurate across different regions.
- **Responsive Design**: Creating a seamless user experience across various devices and screen sizes for the web application.
- **User Interface Design**: Designing an intuitive and user-friendly interface that effectively communicates complex EV information.
- **Integration of Multiple Features**: Coordinating different calculators and tools to work cohesively within a single application.
- **Performance Optimization**: Ensuring fast load times and smooth interactions, especially for calculation-heavy features.
